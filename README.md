# FFPR
Fractal Floating-Point Representation for stable deep learning
# FFPR — Фрактальное представление чисел с плавающей запятой

**Fractal Floating-Point Representation** — инженерное расширение IEEE 754 для детерминированной обработки NaN и Infinity в глубоком обучении.

## О проекте

В работе предлагается новый подход к работе с сингулярностями (0/0, ∞/∞, vanishing/exploding gradients) с помощью фрактальных генераторов `Fk` и механизма Threshold Promotion.

### Основные идеи
- Фрактальные нули и бесконечности разных порядков
- Детерминированное раскрытие неопределённостей за константное время
- Sparse Fractal State для экономии памяти GPU

## Содержимое репозитория

- `Fractal_numbers` — три файла с работой 
- `stress_test.py` — мои жалкие попытки проверить работу системы
